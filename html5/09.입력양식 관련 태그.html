폼 관련 태그

form태그 내에 submit버튼 클릭시 해당 form안에
작성된 사용자가 입력한 값들을 서버로 전송하면서 요청하는 역할 수행
action속성 : 해당form에 사용자가 입력한 값들을 전달할 서버의 경로
method속성 : 요청 전송 방식을 지정하는 속성(get/post)
get : 요청시 사용자가 입력한 값들이url에 노출
post : 요청시 사용자가 입력한 값들이 url에 노출안됨
form 내의 submit버튼 클릭시 action 지정한 서버로 사용자 입력한 값들이 넘어가면서 요청됨
사용자가 입력한 값들은key=value세트로 전달
