CSS 기본선택자

선택자란?

특정 html요소를 선택하고자 할 때 사용하는 기능
해당 요소를 선택하여 원하는 스타일과 기능을 적용시킬 수 있음

1. 모든 선택자 : *

현재 이 문서의 모든 요소들을 선택하고자 할 때 사용

    *{
        스타일속성 :값;
        스타일송석 :값;
    }
본문에다가 스타일 넣어도 된다.

2. 태그선택자 : 태그명

현재 이 문서의 해당 태그들을 모두 선택하고자 할 때

    태그명{
        스타일속성 :값;
        스타일송석 :값;
    }

3. 아이디선택자 : #아이디명

현재문서에서 특정 요서 하나만을 선택하고자 할 때 사용

4. 클래스선택자 : .클래스명

문서내에서 내가 원하는 요서 여러개 선택하고자 할 때 사용
