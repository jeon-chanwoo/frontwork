1. 속성 선택자

선택하고자 하는 요소내에 작성되어 있는 속성을 이용하여 선택하는 방법
선택자 위에[]를 이용하여 속성과 속성값을 제시하면서 선택

    선택자[속성=속성값]{스타일속성 : 값},속성=속성값일때
    선택자[속성~=속성값] {스타일송석 : 값;}속성값이 단어로 포함되어 있을 때" "로 구분
    선택자[속성|=속성값] {스타일송석 : 값;}속성값이 특정 값을 단어로 포함하고 있을때 "-"로 구분
    선택자[속성^=속성값]{스타일속성 : 값},속성값으로 시작하는 태그 선택
    선택자[속성$=속성값]{스타일속성 : 값},속성값으로 끝나는 태그 선택
    선택자[속성*=속성값]{스타일속성 : 값},속성값이 포함되어 있는 태그 선택
