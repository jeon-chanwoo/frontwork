배열

- 변수 선언시 별도의 자료형 지정하지 않은
- 어떤 자료형의 값들이든 다 하나의 배열 공간에 담을 수 있음, 크기에 제약도 없다(자바의 컬렉션과 유사)
- 존재하지 않는 인덱스를 참조해도 오류나지 않음. undefined리턴

* 배열의 선언

* 배열관련 메소드

1) 배열.indexOf(찾고자하는요소) : 배열에서 해당 요소가 위치의 인덱스를 반환

2) 배열.concat(배열, 배열, ...) : 배열 여러개를 결합하고자 할 때 사용

3) 배열.reverse() : 배열에 담긴 값들을 역순으로 바꿔주는 메소드

4) 배열.sort() : 배열에 담긴 값들을 오름차순으로 정렬시켜주는 메소드

5.1)배열.push: 배열의 맨뒤에 요소를 추가시키고 배열의 크기를 반환 5.2)배열.pop : 배열의 맨뒤의 요소를 제거하고 제거된 요소를 반환

6.1) 배열.unshift(추가할 요소) : 배열의 맨 앞에 요소 추가한 후 배열의 크기 반환<
6.2) 배열.shift() : 배열의 맨앞에 요소 제거하고 제거된 요소 반환


7.1)배열.slice(시작인덱스, 끝인덱스) : 배열에서 시작인덱스부터 끝인덱스까지 추출해주는 메소드
7.2)배열.slice(시작인덱스, 제거수,추가할 값) : 배열의 요소 추출하여 제거 및 추가

8) 배열.toString()/ 배열.join([구분자]) : 배열에 각 인덱스에 담긴 값들을 하나의 문자

9) 반복처리 for문, for..in,for..of,foreach
